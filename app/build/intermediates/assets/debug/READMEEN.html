Tag Read

RFID HF card supports for common standards such as: ISO 14443-3A, ISO 14443-3B, ISO 14443-4, JIS 6319-4, ISO 15693, Mifare Classic, and Mifare Ultralight. It provides a convenient data storage function and can be used for future viewing.

Instructions :

Put the card on the PDA keypad parts. If the device recognizes the card, it will automatically pop up its details. Click on "Review" can browse through all the current scan software card information; click "Save", the current scan results will be permanently saved to the database for viewing the record.

Tag Write

Used to write into cards and the card must have rights to read and write. Note: The write operation will erase the data in the card so the operation need to be cautious.

Instructions :

Firstly select the type of write data, URI types is only required to complete specific data and supports both Chinese and English; MIME type needs to indicate the MIME type; the name should not be more than 255 bytes; contents support both Chinese and English; it can also directly edit raw bytes. For convenient, use decimal to mark the value of each byte; use a minus sign to separate. If you enter any bytes overflow, the overflowed portion will be automatically discarded. Click on the "Write" to wait for the results.

Record Browse

View records previously saved, you can view the card information through the contents and can also clear the saved records.

Instructions :

Record contains the UID (card) and scanning time, according to these two items to choose what you want to check. Click the corresponding items to read details. Click the "Clear" button to clear the current data in the list.
